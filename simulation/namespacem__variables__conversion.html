m_variables_conversion Module Reference

This module features a database of subroutines that allow for the conversion of state variables from one type into another. At this time, the state variables type conversions below are available: 1) Mixture => Mixture 2) Species => Mixture 3) Conservative => Primitive 5) Conservative => Flux 6) Primitive => Conservative 8) Primitive => Flux. Detailed Description

This module features a database of subroutines that allow for the conversion of state variables from one type into another. At this time, the state variables type conversions below are available: 1) Mixture => Mixture 2) Species => Mixture 3) Conservative => Primitive 5) Conservative => Flux 6) Primitive => Conservative 8) Primitive => Flux.
